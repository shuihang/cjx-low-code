# I18n

The Cjx Low Code component defaults to using Chinese. If you wish to use other languages, you can refer to the following solution.

## Global Configuration

The Cjx Low Code component defaults to using Chinese. If you wish to use other languages, you can refer to the following solution.

```ts
// main.ts
import CjxLowCode from 'cjx-low-code'

app.use(CjxLowCode, {
  locale: 'en',
})
```
